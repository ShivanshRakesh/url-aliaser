URL Aliaser

 Give alias to frequently visited or long URLs.

Usage

  • Click on Manage Aliases to Add/Remove aliases.
  • Click on    to ADD and    to DELETE an alias entry.
  • To use an alias, type a registered alias in the address bar followed by a DOUBLE SLASH (//).
  • Example: If the alias for www.gmail.com is set as mail, then on the address bar type mail//.
