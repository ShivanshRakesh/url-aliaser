URL Aliaser

 Give alias to frequently visited or long URLs.

Usage

  • Click on Manage Aliases to Add/Remove aliases.
  • Click on    to ADD and    to DELETE an alias entry.
  • To use an alias, type a registered alias in the address bar followed by a FORWARD SLASH (/).
  • Example:  If the alias for www.google.com is set as g, then in the address bar type g/.
  • PLACEHOLDERS:
  • In the url entry, use DOUBLE PERCENTAGE SIGN (%%) to specify a placeholder.
  • To use a placeholder, type the substitutions for respective placeholders separated by FORWARD SLASH (/).
  • Do append a FORWARD SLASH (/) to whatever entered in the address bar.
  • Example:  If the alias for xyz/%%/a1/%%/a2 is set as test, then in the address bar type test/plc1/plc2/ to redirect to xyz/plc1/a1/plc2/a2.
